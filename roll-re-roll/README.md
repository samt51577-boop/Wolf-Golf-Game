# Roll Re-Roll Golf Game

## How to Run
1. Open your terminal.
2. Navigate to this directory:
   ```bash
   cd /Users/samkimberly-ellie/Desktop/Ideas/roll-re-roll
   ```
3. Start the local server:
   ```bash
   python3 -m http.server 8080
   ```
4. Open your browser to: [http://localhost:8080](http://localhost:8080)

## Features
- **All 50 US States** included in course selection.
- **Match Play** with Roll / Re-Roll / Press mechanics.
- **Umbrella Game** (Points).
- **Custom Course** creation.
